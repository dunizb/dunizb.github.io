jQuery Core Style Guide（规范）

Google JavaScript Style Guide定义了如下规范：

  • 除了意图明确的需要创建一个全局变量之外，总是使用var声明变量；
  • 总是使用分号。只对于压缩代码非常重要；
  • 常量使用大写，每个单词之间用一个下划线分隔；
  • 函数、变量和方法名都是用驼峰写法，但整个变量名第一个字母小写；
  • 类和枚举名也采用驼峰写法，但整个变量名的第一个字母大写。
